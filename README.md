# 🗳️ Democra-cy

A full-stack web application exploring the concept of **digital democracy**, built using modern web technologies with real-time voting, authentication, and performance optimizations.

---

## 🌐 Live Demo
https://democra-cy.vercel.app/

---

## 🛠️ Tech Stack

- **Next.js (App Router)** – SSR & routing  
- **React.js** – Component-based UI  
- **Tailwind CSS** – Responsive styling  
- **Firebase** – Authentication, Firestore DB, Analytics  
- **Vercel** – Deployment & CI/CD  

---

## ⚙️ Features

- 🔐 Google Authentication (Firebase Auth)  
- 🗳️ Real-time Voting System  
- ☁️ Firestore Database Integration  
- ⚡ Optimized performance (lazy loading, modular code)  
- 🎯 Clean and responsive UI  

---

## 🔐 Security Enhancements

- XSS protection using input sanitization  
- Input validation (email & password)  
- Content Security Policy (CSP) implemented  
- Secure Firebase integration  

---

## ⚡ Performance Optimization

- Spatial partitioning (O(n) particle system)  
- Lazy loading with dynamic imports  
- Optimized image handling using Next.js  

---

## ♿ Accessibility Improvements

- Semantic HTML structure  
- Accessible forms with labels  
- Keyboard navigation support  
- ARIA attributes for assistive technologies  

---

## 🧪 Testing

- Basic unit testing using Jest & React Testing Library  
- Component rendering tests
