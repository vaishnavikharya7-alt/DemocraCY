# 🚀 Democra-cy

A modern full-stack web application exploring the concept of **digital democracy**, built with a focus on performance, scalability, and clean user experience.

## 🌐 Live Demo
https://democra-cy.vercel.app/

---

## 🛠️ Tech Stack

- **Next.js** – Server-side rendering & routing  
- **React.js** – Component-based UI development  
- **Tailwind CSS** – Utility-first styling  
- **Vercel** – Deployment & CI/CD  

---

## ⚙️ Features

- ⚡ Fast performance with SSR & Static Generation  
- 🎯 Clean and responsive UI/UX  
- 🧩 Scalable component-based architecture  
- 🔄 Smooth navigation and optimized loading  
- 🌍 Deployed with global edge network (Vercel)  

---

## 📁 Project Structure
├── components/ # Reusable UI components
├── pages/ or app/ # Routing (Next.js)
├── styles/ # Tailwind & global styles
├── public/ # Static assets
└── utils/ # Helper functions
